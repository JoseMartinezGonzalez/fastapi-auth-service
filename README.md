**Title:**
FastAPI Microservice for User Authentication & Role-Based Access

**Description:**
A production-ready FastAPI microservice that handles user registration, login, JWT authentication, role-based permissions, and secure API endpoints. Designed with clean architecture, async support, and full error handling.
This example shows how I build real-world backend services that are fast, scalable, and easy to integrate with any frontend or other services.

**Technologies:**
•	FastAPI
•	Python
•	JWT (PyJWT)
•	SQLAlchemy
•	SQLite/PostgreSQL
•	Pydantic
•	Uvicorn

**Key Features:**
•	User registration + login
•	JWT-based authentication
•	Admin/User role permissions
•	Async endpoints
•	Centralized error handling
•	Reusable service layer
•	Built-in unit tests
•	Clean folder structure
